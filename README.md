# League of Legends Match Data Exploration

Small data-cleaning and modeling project for exploring interest in AI & data science.  
Dataset: https://www.kaggle.com/datasets/californianbill/patch-25-14-lol-league-of-legends-ranked-games/data

Goals:

- Learn Python data wrangling and cleaning
- Practice exploratory analysis on real game data
- Build a simple logistic regression model to predict match wins

Key tools: pandas, scikit-learn, matplotlib
